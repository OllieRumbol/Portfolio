React is a simple but powerful JS library used to create single-page web applications. React is built by Facebook and full documentation can be found here. The React library provides a whole host of useful tools. Tools include but not limited to JSX, virtual DOM, one-way binding, state management and component design.

This article illustrates some basic steps developers new to react can take to ensure their React project has a solid base for future development. I will show how the React app can be deployed using GitHub pages as well as some other useful tools. Before starting anything its a good idea to create a GitHub repo and clone it locally
